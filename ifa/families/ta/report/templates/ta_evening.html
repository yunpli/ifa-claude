§00

本报告是什么 · 不是什么

✓ 本报告是

  • 3 周 swing 选股器。每个候选股的目标持仓 ≤ 15 个交易日(约 3 周),由"技术失效位 / 形态目标区 / 时间到期"三重出场机制管理。
  • 单股目标 +6-9%(基于 ATR × k_target=3.0,约 2:1 R:R)。Tier A 历史表现:在同周期所有候选股中,T+15 平均收益领先 0.8-1.2pp。
  • 技术形态识别:基于 30 个量化技术 setup(突破/回踩/反转/形态/量价/板块共振/筹码/主力资金/顶部反转/统计极值/事件催化),叠加体制感知评分。
  • Tier A/B 是相对排名:Tier A 是当日 ~3000 候选中得分最高的 ~10 只,代表"短期 setup 强度 + 板块共振 + 历史胜率"综合最优。

✗ 本报告不是

  • 不是长持高收益策略。任何"40 天 +50%"或"年化 100%"的预期都不在本系统设计目标内。要长持龙头/超级强势股,需另行使用基本面 + 趋势跟踪框架。
  • 不是个股推荐。Tier A/B 是统计排名,不是确定性买入信号。任何单只票都可能因突发事件 / 黑天鹅 / 流动性丧失而失败。
  • 不是大盘 alpha。"vs 平均"指的是 vs 同周期所有命中候选的均值,不是 vs 沪深 300。系统衡量的是"内部 selectivity edge"。
  • 不替代风控。建议仓位 ≤ 5%/单、≤ 30%/同板块,严格遵守技术失效位规则。系统不预测个股暴雷或政策黑天鹅。

技术细节见 docs/ta-strategy-deep-dive.md · 调参与 alpha 来源见 docs/ta-tuning-playbook.md。 历史回测口径:T+15 平均实现收益(state machine: stop/target/time-exit),benchmark 为同窗口所有 filled 候选的等权平均。

说明

{{ s.title }}

系统全市场扫描 30 个独立策略（分 11 族），每个策略独立判定个股是否命中。 同一支股票若被多个策略同时命中，**多策略共振 = 高 conviction 信号**。 每个候选股进一步基于 ATR 计算入池观察价区 / 技术失效位 / T+15 观察目标区，供参考使用。

D 顶部反转（警示）双顶 · 头肩顶 · 流星线（不进 Tier A/B，§13 风险扫描）
Z 统计极端 Z-score · 超卖反弹 · 横盘 fade-rally
E 事件业绩预告 / 快报 / 披露窗口催化

综合分计算（决定星级、决定是否进 Tier A/B）：

  • 基础 取该股所有命中策略中的最高分（trigger 累加 + 连续体制加成 + 胜率折算） MAX(scores)
  • Bayesian 共振 次族 setup 按各自胜率加权累加（高胜率族贡献大），最多 3 个次族贡献 +0.08/0.05/0.03
  • 体制连续加成 基于该策略当前体制 60d 胜率 vs 整体胜率比值的连续 boost [-0.20, +0.50]
  • 边际自调 基于该策略近 60 日 combined T+15 边际自动加权，弱边际 setup 自动降权 × [0.80, 1.20]
  • 板块乘子 SmartMoney 板块净流入排名 + cycle phase + 置信度合成的板块质量乘子 × [0.60, 1.00]
  • 胜率折算 按策略近 60 日 T+15 实际胜率打折（30% 不打折、15% 折半、地板 40%） × 胜率/30%
  • 衰减治理 策略 60d-250d 胜率差 ≤ -10pp 仅观察、≤ -15pp 整段剔除 M5.3 闸门

星级映射（综合分 0.000-99.999 区间）： ≥85.00 = ★★★★★（高 conviction）· ≥75.00 = ★★★★ · ≥65.00 = ★★★ · ≥55.00 = ★★ · 其余 = ★。
分层池：按综合分排序后切片：Tier A 重点 (10) / B 候选 (20) / C 观察 (100) / 其余丢弃。 多策略同股 → tie-break 优先取共振更高（族数更多）。
